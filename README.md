## 3. 실행 방법

### 3.1 Backend
- **git clone** 및 **`setup.py`** 파일을 통해 설치합니다.
```
$ git clone https://github.com/Juhee-Maeng/Movie_Search.git
```
```
$ cd Movie_Search/movie
$ python setup.py install
```
- **movie** 명령어를 통해 실행합니다.
```
$ movie
```

### 3.2 Frontend
- **npm**을 통해 설치합니다.
```
$ cd Movie_Search/front-end
$ npm install
```
```
$ npm start
```
